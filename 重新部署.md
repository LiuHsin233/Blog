# 重新部署步骤(比如更换电脑之类的)
1. 使用git克隆对应的项目 `https://github.com/LiuHsin233/Blog.git`
2. 安装hexo等工具(使用git bash here或者终端都可以,注意好对应的位置即可).
   > 在项目根目录,这里是Blog\hexo\目录下使用git bash
   > 分别输入`npm install hexo`,`npm install`,`npm install hexo-deployer-git`  
   > 注意 这里不需要运行hexo init(因为不是空文件夹)
3. (可选)将主题克隆至对应的项目目录中`git clone https://github.com/ppoffice/hexo-theme-icarus themes/icarus`
   > 如果不确定可以先把主题克隆到其他目录然后拷贝过去
   > 主题这块后续不会被提交,建议额外保存下地址方便后续操作
4. 使用`hexo g`和`hexo s`测试,本地生成顺利就表示可以了(如果还有问题可以用询问ai,这样会比较快)
